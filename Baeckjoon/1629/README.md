# [백준] 1629번 : 곱셈
- https://www.acmicpc.net/problem/1629
- 거듭제곱을 구현하는 문제이다. a를 b번 곱해서 c로 나눈 것을 구하는 문제이다.
  - 단순히 계속 거듭제곱을 하면 숫자의 범위가 너무커지므로 재귀적으로 b/2번 곱한 수를 구해서 곱하는 문제이다.
  - 이 때 b가 홀수라면 (a*(b/2번 제곱한 수)*(b/2번 제곱한 수))%c 해서 a를 한번 더 곱해줘야 한다. (ex. a^5 = (a*(a^2%c)*(a^2%c))%c)
  - 재귀가 멈추는 조건은 b가 1이 되면 a%c를 반환하는 것이다.(b는 자연수이고, 홀수일 때 짝수로 맞춰서 호출하기 때문)
