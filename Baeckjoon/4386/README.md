# [백준] 4386번 : 별자리만들기
- https://www.acmicpc.net/problem/4386
- MST를 만드는 문제이다.
- 이 문제를 푸는 방법은 프림알고리즘, 크루스칼알고리즘이 있다.
  - 그래프 내에 간선이 많은 밀집그래프 -> 프림알고리즘
  - 그래프 내에 간선이 적은 희소그래프 -> 크루스칼알고리즘
- 내 경우에는 크루스칼알고리즘이 익숙해서 크루스칼로 풀었다.
  1. 모든간선을 구한다.
  2. 오름차순으로 정렬한다.
  3. 유니온파인드를 실행한다.
- 이 문제의 경우 소수로 입력받아야 하기 때문에 이 부분을 몰랐었다.
  - 'cout<<fixed;'로 소수점을 고정시키고, 'cout.precision(2)' 로 소수점2째자리까지 표현하는 것을 선언한다.
- 이 문제에서 노드의 경우 결국 1~n까지의 노드가 존재한다고 볼 수 있다. 즉 소수가 아닌 정수로 저장이 가능하다.
  - 거리만 double로 실수로 저장하고, 노드의 경우 각 인덱스로 변환해서 저장하면 된다.
