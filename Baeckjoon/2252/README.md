# BOJ 줄 세우기
- https://www.acmicpc.net/problem/2252
- 위상정렬 문제다. 위상정렬이란 어떤일의 순서를 찾는 알고리즘이다. 각 정점들의 선행순서를 지키면서 모든 정점을 나열하는 것이다.
	1. 진입차수가 0인 모든 정점을 큐에 삽입
	2. 큐가 빌때까지 다음과정을 반복
		1. 큐에서 정점을 꺼내서 해당정점에서 나가는 간선을 그래프에서 제거 
		2. 새롭게 진입차수가 0인 정점을 큐에 삽입
- 위상정렬에 대해서 새롭게 공부를 하게되서 좋은 문제였다. 이 정렬의 특징은 사이클이 존재하는 그래프에서는 수행할 수 없다는 것이다. 또한 정답이 여러개일 수 있다.
