# [백준] 2213번 : 트리의 독립집합
- https://www.acmicpc.net/problem/2213
- 트리에서의 DP문제이다.
  - 독립집합이라고 길게 설명되어 있지만 간단히 설명하면 두가지의 경우를 고려해야 하는 것이다.
    - 현재노드가 포함되어 있을 때 : 직접 연결되어 있는 노드들은 집합에 포함되지 않는다.
    - 현재노드가 포함되어 있지 않을 때 : 자식노드가 독립집합에 포함된경우, 자식노드가 독립집합에 포함되지 않은 경우
  - 즉 위 두가지 경우에서 최대값을 선택하면 되는 것이다.
  - 독립집합의 수만 찾는 것이 아니라 그에 속한 노드들도 출력해야 되기때문에 dfs로 크기를 구한 후 포함된 각 노드들을 탐색을 진행했다.
  - 늘 DP문제가 그렇듯 점화식을 어떻게 유도하느냐가 중요한 문제였다. 이 문제정도면 무난했다고 생각한다.
