# [백준] 2887번 : 행성터널
- https://www.acmicpc.net/problem/2887
- MST를 구성하는 문제이다.
- 생각보다 많이 어려운 문제였다. n의 크기가 엄청 크니까 n^2을 하게되면 메모리 초과가 나는 문제이다. 즉 크루스칼방식으로 모든 간선을 찾기는 안된다는 것이다.
 - 가중치를 구하는 방법은 x,y,z중에 차이가 가장 적은 것이니까 x,y,z각각을 오름차순으로 정렬해서 간선벡터에 넣어준 뒤, 가중치기준으로 정렬하면 된다.
 - 여기서 문제는 기존의 c++의 sort을 쓸려면 x,y,z를 담은 struct에 담아서 사용해야되는데 struct를 sort하기 위해 오버로딩해서 비교연산을 지정해줘야된다.
 - 가중치까지 다 구했으면 기존의 크루스칼알고리즘대로 풀면된다.
- 이 문제의 핵심은 간선을 어떻게 n^2이 아닌 방법으로 구할것이냐는 것이다. 생각보다 많이 어려웠고, 이후에 다시한번 풀어봐야할 문제같다.
