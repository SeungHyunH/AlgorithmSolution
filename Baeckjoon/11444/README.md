# [백준] 11444번 : 피보나치 수 6
- https://www.acmicpc.net/problem/11444
- 이 문제는 피보나치의 점화식을 행렬로 표현해야 한다.
  - 피모나치 점화식 : Fn+2 = Fn+1 + Fn
  - 이 것을 {Fn+1,Fn} = {{1,1},{1,0}}^(n-1) * {F2,F1} 이렇게 표현이 가능하다.
    - https://seokjin2.tistory.com/11 이 블로그를 참조해서 이해했다. 그림을 통해 자세히 설명이 되어있다.
  - 이제 피보나치점화식을 행렬의 거듭제곱으로 표현할 수 있으니 거듭제곱을 분할정복을 통해 구현하면 된다.
