# [백준] 2618번 : 경찰차
- https://www.acmicpc.net/problem/2618
- DP로 푸는 문제이다.
  - 이번 문제는 DP를 가장 최근에 처리한 사건의 인덱스와 거리를 저장하면 된다. 그렇게되면 경찰차1의 마지막위치 = 경찰차1이 마지막으로 처리한 사건이 된다.
  - DP`[A][B]`= C 라고 했을 때 A는 경찰차1이 가장 최근에 처리한 사건, B는 경찰차2가 가장 최근에 처리한 사건, C는 그 두 사건을 처리했을 때 이동거리의 최솟값이라고 정의한다.
  - 이렇게 두는 이유는 A와B중 큰 값은 가장 최근에 처리한 사건이 된다. 그랬을 때 다음사건은 max(A,B)+1 가 된다.
  - 점화식을 각각의 경찰차가 사건을 해결했을 때 경로값 + 앞으로의 경로값 중 최소값 이라고 정하면 재귀적으로 맨마지막사건을 처리하고나면 한꺼번에 종합해서 최솟값을 구할 수 있게 된다.
- 설명하기가 까다롭고, 이해하기도 어려운 문제였다. 특히 DP로 어떤 값을 저장해야하는 지가 생각하기 어려운 문제였다. 다시 풀어봐야 할 필요성이 있는 문제다.
