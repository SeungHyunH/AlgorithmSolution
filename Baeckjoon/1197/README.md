# [백준] 1197번 : 최소 스패닝 트리
- https://www.acmicpc.net/problem/1197
- MST를 구현하는 문제이다.
  - MST(Minimum Spanning Tree) : 모든 정점이 연결하는 부분그래프 중 가중치의 합이 최소인 트리
- 유니온파인드를 이용해 트리를 구성하면 된다.
  1. 가중치를 기준으로 정렬을 한다.
  2. 가중치가 제일 낮은것부터 트리를 구성한다.
    - 이 때 유니온 파인드를 이용해 사이클이 형성되지 않게 만든다.
- 알면 쉽지만 모르면 구현하기 어려운 문제... 역시 알고리즘은 계속 공부해야함을 깨닫는다.
- 유니온파인드를 까먹었다면 https://www.acmicpc.net/problem/1717 을 참조하자
