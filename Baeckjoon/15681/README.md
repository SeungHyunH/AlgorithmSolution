# [백준] 15681번 : 트리와 쿼리
- https://www.acmicpc.net/problem/15681
- 트리와 임의의 정점u가 주어졌을 때, u의 서브트리에 있는 정점의 개수를 구하는 문제이다.
- 트리의 DP문제로 분류가 되어있고, 장황하게 설명했지만 결국 간단한 dfs문제이다. 각 서브트리를 조회하면서 size크기 n을 리턴받으면 되는 구조이다.
  - 당연히 임의의 정점 u가 여러개가 주어지기 때문에 일일이 그때마다 dfs를 할 수는 없으니 dp에 저장해서 출력하면 된다. 아마 dp로 분류된 이유는 이 것 때문인듯 하다.
