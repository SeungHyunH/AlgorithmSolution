# [백준] 2533번 : 사회망 서비스(SNS)
- https://www.acmicpc.net/problem/2533
- 전형적인 dp문제이다. 점화식을 세우면 된다.
  - 두가지의 경우가 있다.
  1. 현재 노드가 얼리어답터일때 : 자식 노드가 얼리어답터이거나 아닌경우 둘중 작은값을 선택해 더하면 된다.
  2. 현재 노드가 얼리어답터가 아닐 때 : 자식 노드가 얼리어답터일때의 경우를 더한다.
  - dfs로 서칭을 하면서 위 점화식을 찾으면 되는 문제이다.
- dp와 트리서치 등을 물어볼 수 있는 전형적인 문제이므로 주의할 필요가 있는 문제같다.
