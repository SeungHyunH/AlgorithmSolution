# [백준] 9372번 : 상근이의 여행
- https://www.acmicpc.net/problem/9372
- 이 문제는 MST의 특성을 물어보는 문제였다.
- 이 문제를 보자마자 떠오르는 생각
  1. BFS나 DFS로 최단거리를 찾아야되나?
  2. 카테고리가 최소 신장 트리니까 MST에 관해서 추가적으로 공부해서 풀어봐야 되나?
- 결론은 조건에 답이 있었다.
  1. 항상 연결그래프를 이룬다.
  2. 가중치가 없다.
  - N개의 노드를 연결하는 최소간선의 수를 구하면된다. -> 즉 답이 N-1이다.
- N-1만 출력하면되는 어이없는 문제였지만 이것이 출제자의 의도라고 한다.(놀라울 따름..)
- 가중치가 주어졌다면 프림알고리즘이나 DFS,BFS로 찾아야 될것이다.
